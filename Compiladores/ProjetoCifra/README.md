# 🔐 Projeto Cifras Clássicas

## 📌 Descrição

Este projeto foi desenvolvido com o objetivo de implementar e demonstrar o funcionamento de algoritmos de criptografia clássica estudados na disciplina de Compiladores.

A aplicação permite cifrar e decifrar mensagens utilizando diferentes métodos, explorando conceitos fundamentais de segurança da informação e manipulação de dados.

## 🚀 Tecnologias utilizadas

* Java
* Programação Orientada a Objetos (POO)

## 🔑 Funcionalidades

* 🔒 Cifra de César
* 🔑 Cifra de Vigenère
* ⚡ Cifra RC4
* 🔄 Criptografia e descriptografia de mensagens

## 📚 Conceitos abordados

* Algoritmos de criptografia clássica
* Manipulação de strings
* Estruturas de repetição e controle
* Lógica de programação

## 🎯 Objetivo do projeto

Consolidar conhecimentos em programação e entender na prática como funcionam métodos básicos de criptografia, além de desenvolver organização de código e boas práticas.

## 📌 Observações

Este projeto tem fins educacionais e não deve ser utilizado para segurança real, pois os algoritmos implementados são considerados inseguros para aplicações modernas.
